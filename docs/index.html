HMB Sailing Log

Personal sailing logbook for Android. GPS tracking, nautical maps, PDF export and safety tools.

⬇ Download APK GitHub
🗺️

Nautical Maps

Real-time position on satellite + OpenSeaMap overlay. Seamarks, wrecks, buoys and shallows.

📍

GPS Tracking

Automatic logbook entries with position, speed, course and wind. Track runs in the background.

📄

PDF Export

Professional PDF logbook with map screenshots, entries, skipper signature and QR verification.

🧑‍✈️

Skipper Profile

Store skipper license, VHF/SRC certificate and other credentials. Included in PDF export.

🌤️

Marine Weather

Wind, waves, temperature and conditions from Open-Meteo. Works offline with last cached forecast.

🚨

Safety Tools

MOB alarm, anchor watch, Mayday DSC card, COLREG rules reference and safety briefing checklist.

⚓

Anchor Watch

Set anchor perimeter radius. Vibration alarm if the vessel drifts outside the allowed circle.

📡

Marine Instruments

Connect to Raymarine (WiFi gateway) via TCP or UDP. Live depth, wind, water temp and compass.

🔒

Private & Offline

All data stored locally. No account required. Optional cloud sync to logbook.hmba.boats.
